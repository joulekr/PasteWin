## 1. 설치 및 시작

### 설치

1. [GitHub Releases](https://github.com/joulekr/PasteWin/releases/latest) 에서 최신 `.exe` 설치 파일을 다운로드합니다.
2. 파일을 실행하고 안내에 따라 설치합니다.
3. 설치 완료 후 PasteWin이 자동으로 실행되며 시스템 트레이에 아이콘이 표시됩니다.

### 첫 실행

- 앱은 **시스템 트레이에 숨어** 백그라운드에서 실행됩니다.
- 창을 열려면 `Ctrl+Shift+V` 를 누르거나 트레이 아이콘을 클릭합니다.
- 설정에서 **Windows 시작 시 자동 실행**을 켜두면 매번 수동으로 실행할 필요가 없습니다.

## 2. 기본 사용법

### 창 열기 / 닫기

| 동작 | 방법 |
|------|------|
| 창 열기 | `Ctrl+Shift+V` (기본값, 변경 가능) |
| 창 닫기 | `Escape` 또는 창 바깥 클릭 |

창을 열면 마지막으로 복사한 항목부터 최신 순으로 표시됩니다.

### 항목 붙여넣기

항목을 **더블클릭**하거나 선택 후 `Enter`를 누르면:

1. 해당 내용이 클립보드에 복사됩니다.
2. **자동 붙여넣기**가 켜진 경우 이전에 포커스했던 앱으로 즉시 `Ctrl+V`가 전송됩니다.
3. 자동 붙여넣기가 꺼진 경우 클립보드에만 복사됩니다 — 원하는 위치에서 직접 `Ctrl+V`를 누르세요.

> **팁** — PasteWin 창이 열리기 직전에 작업 중이던 앱에 포커스를 유지한 채 단축키를 누르면 자동 붙여넣기가 더 정확하게 동작합니다.

## 3. 클립보드 항목

PasteWin은 다음 4가지 타입의 클립보드 내용을 자동으로 수집합니다.

| 아이콘 | 타입 | 수집 조건 |
|--------|------|-----------|
| T | **텍스트** | 일반 텍스트 복사 |
| 🖼 | **이미지** | 이미지 복사 (스크린샷 포함) |
| 📁 | **파일** | 파일 탐색기에서 파일 복사 |
| 🔗 | **URL** | `http://` 또는 `https://`로 시작하는 텍스트 |

### 항목 카드 정보

각 카드에는 다음 정보가 표시됩니다.

- **내용 미리보기** — 텍스트는 앞부분, 이미지는 썸네일
- **출처 앱 아이콘** — 복사한 앱의 아이콘
- **복사 시각** — 몇 분 전 / 날짜
- **텍스트 길이** (텍스트·URL) / **이미지 크기** (이미지)

### URL 항목

URL로 인식된 항목은 더블클릭하면 붙여넣기 대신 **기본 브라우저로 바로 열립니다**.

### 중복 처리

동일한 내용을 다시 복사하면 새 항목이 추가되지 않고 기존 항목의 **복사 시각만 최신으로 갱신**됩니다. 목록이 지저분해지지 않습니다.

---

## 4. 검색 및 필터

### 검색

상단 검색창에 키워드를 입력하면 실시간으로 항목을 필터링합니다.

- 텍스트·URL은 **전체 내용** 검색
- 이미지·파일은 **미리보기 텍스트** 검색
- `Ctrl+F`로 검색창에 빠르게 포커스할 수 있습니다.
- `Escape`로 검색어를 지우고 검색창에서 포커스를 해제합니다.

### 타입 필터

검색창 옆 칩 버튼으로 타입별로 필터링합니다.

| 버튼 | 표시 항목 |
|------|-----------|
| 전체 | 모든 항목 |
| T | 텍스트만 |
| 🖼 | 이미지만 |
| 📁 | 파일만 |
| 🔗 | URL만 |

---

## 5. 핀 고정 및 단축키 붙여넣기

### 핀 고정

자주 사용하는 항목을 목록 상단에 고정할 수 있습니다.

- 항목 카드 우측의 **핀 아이콘**을 클릭해 고정/해제합니다.
- 핀 항목은 목록 최상단 **핀 섹션**에 별도로 표시됩니다.
- 핀 항목은 **자동 정리 대상에서 제외**됩니다.

### 핀 순서 변경

핀 섹션의 항목을 **마우스로 드래그**해 원하는 순서로 재정렬할 수 있습니다.

### 전역 단축키로 즉시 붙여넣기

핀된 항목을 `Ctrl+Shift+숫자`로 PasteWin 창을 열지 않고 즉시 붙여넣을 수 있습니다.

| 단축키 | 동작 |
|--------|------|
| `Ctrl+Shift+1` | 핀 첫 번째 항목 즉시 붙여넣기 |
| `Ctrl+Shift+2` | 핀 두 번째 항목 즉시 붙여넣기 |
| … | … |
| `Ctrl+Shift+9` | 핀 아홉 번째 항목 즉시 붙여넣기 |

> **팁** — 자주 쓰는 텍스트 템플릿, 이메일 서명, 코드 스니펫을 핀으로 고정해두면 단축키 한 번으로 어디서든 붙여넣을 수 있습니다.

## 6. 다중 선택

여러 항목을 동시에 처리할 수 있습니다.

| 동작 | 방법 |
|------|------|
| 항목 선택/해제 | 클릭 |
| 추가 선택 | `Ctrl+Click` |
| 전체 선택 | `Ctrl+A` |
| 선택 항목 붙여넣기 | `Enter` |
| 선택 항목 삭제 | `Delete` |

다중 선택 후 `Enter`를 누르면 선택된 항목이 **순서대로 연속 붙여넣기**됩니다.

---

## 7. 텍스트 편집

저장된 텍스트·URL 항목의 내용을 직접 수정할 수 있습니다.

1. 항목 카드에 마우스를 올리면 우측에 **편집(✏) 버튼**이 나타납니다.
2. 버튼을 클릭하면 전체화면 편집 모달이 열립니다.
3. 내용을 수정한 후 **저장**을 클릭합니다.

> 이미지·파일 항목은 편집이 지원되지 않습니다.

---

## 8. 이미지 미리보기

이미지 항목을 더 크게 확인할 수 있습니다.

1. 이미지 카드에 마우스를 올리면 우측에 **미리보기(🔍) 버튼**이 나타납니다.
2. 버튼을 클릭하면 별도 창에서 원본 크기로 이미지가 열립니다.
3. 미리보기 창에서 **복사 버튼**을 클릭하면 클립보드에 이미지가 복사됩니다.

## 9. 일시정지

클립보드 수집을 잠시 멈출 수 있습니다.

### 일시정지 방법

- TopBar 상단의 **⏸ 버튼** 클릭
- 시스템 트레이 우클릭 → **일시정지** 메뉴

### 일시정지 중

- 새로 복사한 내용이 PasteWin에 저장되지 않습니다.
- ⏸ 버튼이 강조 표시되고 상태바에 `일시정지 중` 메시지가 표시됩니다.
- 비밀번호·보안 정보 복사 시 임시로 켜두면 유용합니다.

### 재개

동일한 버튼(▶)을 다시 클릭하거나 트레이 메뉴에서 **재개**를 선택합니다.

**일시정지 상태는 앱을 재시작해도 유지됩니다.**

## 10. 시스템 트레이

트레이 아이콘을 **우클릭**하면 메뉴가 표시됩니다.

| 메뉴 항목 | 동작 |
|-----------|------|
| PasteWin 열기 | 메인 창 표시 |
| 일시정지 / 재개 | 클립보드 수집 토글 |
| 전체 삭제 | 핀 제외 전체 기록 삭제 |
| 설정 | 설정 창 열기 |
| 정보 | 버전·업데이트 확인 창 열기 |
| 종료 | 앱 완전 종료 |

## 11. 설정

상단 **⚙ 설정 버튼** 또는 트레이 메뉴 → 설정으로 열 수 있습니다.

### 일반

| 설정 | 설명 |
|------|------|
| **자동 붙여넣기** | 항목 선택 시 이전 앱으로 `Ctrl+V` 자동 전송 |
| **Windows 시작 시 자동 실행** | 로그인 후 PasteWin 자동 시작 |

### 기록 관리

| 설정 | 설명 | 기본값 |
|------|------|--------|
| **최대 보관 수** | 히스토리 최대 항목 수 (10~9999) | 500 |
| **자동 삭제 기간** | N일이 지난 항목 자동 삭제, 0이면 비활성화 | 30 |

> 핀 고정된 항목은 최대 보관 수·기간 제한을 받지 않습니다.

### 화면

| 설정 | 옵션 |
|------|------|
| **레이아웃** | 가로 (화면 하단 고정 바) / 세로 (화면 우측 패널) |
| **언어** | 한국어 / English |
| **테마** | 다크 / 라이트 / 시스템 설정 따름 |

### 단축키

창 열기/닫기에 사용할 전역 단축키를 변경할 수 있습니다.

1. 단축키 입력란을 **클릭**합니다.
2. 원하는 키 조합을 **누릅니다** (예: `Ctrl+Alt+C`).
3. 키에서 손을 떼면 자동으로 등록됩니다.
4. `Escape`를 누르면 변경 취소입니다.

> 수정키(Ctrl / Alt / Shift / Win) 없이 단독 키만 누르는 것은 허용되지 않습니다.

### 수집 타입

텍스트·이미지·파일·URL 각각의 수집을 개별적으로 켜고 끌 수 있습니다.  
비활성화된 타입은 수집되지 않으며 목록 필터에서도 숨겨집니다.

### 앱 블랙리스트

특정 앱에서 복사한 내용을 수집하지 않도록 설정합니다.

**추가 방법:**

1. 입력란에 앱 실행 파일의 **전체 경로 또는 파일명 일부**를 입력합니다.
   - 예시: `KeePass.exe`, `1Password`, `C:\Program Files\...`
2. `추가` 버튼 또는 `Enter`를 눌러 목록에 추가합니다.

비밀번호 관리자, 보안 메모 앱 등을 등록해두면 민감한 정보가 클립보드 기록에 남지 않습니다.

## 12. 자동 업데이트

PasteWin은 앱 시작 후 백그라운드에서 자동으로 업데이트를 확인합니다.

### 업데이트 알림

새 버전이 있으면 메인 창 하단에 **업데이트 배너**가 표시됩니다.

```
🔔 새 버전 X.Y.Z 업데이트 가능   [업데이트]  [나중에]
```

### 설치 방법

1. 배너의 **업데이트** 버튼을 클릭합니다.
2. 다운로드 진행률이 실시간으로 표시됩니다.
3. 다운로드 완료 후 설치가 진행되며 앱이 자동으로 재시작됩니다.

### 나중에

**나중에** 버튼을 클릭하면 배너가 닫힙니다.  
다음 번 앱 시작 시 다시 확인합니다.

### 수동 확인

상단 **ℹ 정보 버튼** → **업데이트 확인** 버튼으로 언제든지 직접 확인할 수 있습니다.

### 보안

모든 업데이트 파일은 **minisign 서명**으로 검증됩니다.  
서명이 일치하지 않으면 설치가 차단됩니다.

## 13. 단축키 전체 목록

### 전역 (창이 닫혀 있어도 동작)

| 단축키 | 동작 |
|--------|------|
| `Ctrl+Shift+V` | 창 열기 / 닫기 (기본값, 변경 가능) |
| `Ctrl+Shift+1` ~ `9` | 핀 항목 즉시 붙여넣기 |

### 창 내부

| 단축키 | 동작 |
|--------|------|
| `Enter` | 선택/포커스된 항목 붙여넣기 |
| `Escape` | 검색어 있으면 지우기 → 창 닫기 |
| `Ctrl+F` | 검색창 포커스 |
| `Ctrl+A` | 전체 선택 |
| `Ctrl+Click` | 다중 선택 추가/해제 |
| `Delete` | 선택 항목 삭제 (핀 제외) |
| `Ctrl+Delete` | 현재 필터 기준 전체 삭제 확인 |
| `←` / `↑` | 이전 항목으로 포커스 이동 |
| `→` / `↓` | 다음 항목으로 포커스 이동 |
